# forsenad.nu

Modifierar resedetaljer på sj.se med förseningsstatistik

## What It Does

Denna extension observerar sidan resedetaljer på sj.se. Förseningstatistiken hämtas från www.forsenad-nu

## Installation

### From Source

1. Clone the repository
2. Go to chrome://extensions
3. Enable "Developer mode"
4. Click "Load unpacked"
5. Select the project directory

## Permissions

Behöver inspektera sidan på sj.se. Behöver hämta statistik från forsenad.nu
